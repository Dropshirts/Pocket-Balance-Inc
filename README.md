# 🗄️ Pocket Balance Inc
**A minimalist, local-first personal finance tracker and milestone engine.**

Pocket Balance is a high-performance web application designed for users who want a sleek, private, and fast way to track their "pocket cash," pending inflows, and long-term savings goals.

---

## 💎 Core Features

* **Secure Multi-User System:** Create local accounts with persistent device sessions (no database required).
* **Transaction Ledger:** Track every Inflow and Outflow with automated Taka (৳) formatting.
* **Milestone Engine:** Set savings targets with real-time visual progress bars and completion celebrations.
* **Extras Memo:** A dedicated "pending" ledger for money you are expecting (Incoming) or tasks completed (Done).
* **Adaptive UI:** Modern fintech aesthetic with a smooth sliding Dark/Light mode toggle.
* **Privacy-First:** All data is stored in your browser's `localStorage`. No data ever leaves your device or touches a server.

## 🛠️ Technical Stack

* **Logic:** Pure Vanilla JavaScript (ES6+)
* **Styling:** Modern CSS3 with CSS Variables and Flexbox/Grid
* **Persistence:** Browser LocalStorage API
* **Hosting:** GitHub Pages

## 📂 Installation (For Local Use)

If you want to run this locally instead of on GitHub:
1.  Download the `index.html` file.
2.  Open it in any modern browser (Chrome, Safari, Brave).
3.  Initialize your first Vault with a username and password.

## ⚠️ Security Note
This app is designed for **privacy and speed**. Because it uses `localStorage`, your data is tied to the specific browser and device you use to access the URL. If you clear your browser's "Site Data" or "Cache," your Vault will be reset.
